# AWS Cloud Mastery 1

**Date:** March 14, 2026
**Location:** AWS Vietnam Office (Floor 26), Ho Chi Minh City
**Role:** Attendee (FCJ Cloud Intern - Team NeuraX)

## Event Description

This event was an in-depth knowledge-sharing session for Vietnam's tech community, bringing together developers and cloud enthusiasts to explore practical applications of AI and AWS services through real-world, practice-driven projects.

## Main Activities

The event included 3 technical sessions focused on practical AI and AWS use cases:

**Session 1: Building AI Agents & Frameworks**  
Introduced how to build AI agents using an open SDK framework that automates tool definitions (web search, Python execution) and integrates with models such as Amazon Bedrock. The session also emphasized core agent concepts including memory, multi-step reasoning, and tool calling for complex workflows.

**Session 2: Prompt Engineering & Serverless Architecture**  
Explored prompt engineering techniques (Role Prompting, Chain of Thought, Tree of Thought) to improve output quality and reduce token costs. Demonstrated a real "Prompt Optimizer" extension fully built on AWS Serverless services (S3, CloudFront, Cognito, API Gateway, Lambda, DynamoDB, and Bedrock) with near-zero operational cost.

**Session 3: Applying AI to Real Products**  
Showcased practical AI projects. Demo 1: an IoT smart cabinet for club equipment management using Raspberry Pi, AWS IoT Core, S3, DynamoDB, and Amazon Rekognition for face recognition. Demo 2: a personal finance app using Amazon Textract to scan, analyze, and extract receipt information accurately based on context.

## Outcomes

- Agentic AI is a major trend: AI agents can automate complex workflows through iterative reasoning, environment evaluation, and external tool usage.
- Prompt engineering reduces cost: Well-structured prompts and prioritizing English inputs can significantly reduce input/output token cost when calling large AI models.
- Serverless architecture is highly efficient: Combining AWS Lambda, Cognito, S3, and API Gateway delivers a scalable backend without server management, while keeping costs low for AI projects.
- Amazon Rekognition is powerful for vision tasks: It provides strong face and object recognition capabilities, and Custom Labels can be used for domain-specific training in hardware/IoT projects at reasonable cost.
- Amazon Textract improves document understanding: AI-powered context awareness helps distinguish fields such as product names and prices, making it ideal for complex forms like receipts where traditional OCR often fails.
